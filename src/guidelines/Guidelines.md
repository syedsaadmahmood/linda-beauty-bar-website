# Development Guidelines

This folder contains development guidelines and best practices for the Linda's Beauty Bar website project.

## Available Guidelines

- **[Development Guidelines](./Development.md)** - Code style, component structure, and Git workflow
- **[Styling Guidelines](./Styling.md)** - Tailwind CSS usage, color palette, and responsive design
- **[Component Guidelines](./Components.md)** - Component organization, patterns, and reusability

## Quick Reference

### Getting Started
1. Read the [README.md](../../README.md) for project setup
2. Review [SUPABASE_SETUP.md](./SUPABASE_SETUP.md) for database configuration
3. Follow the guidelines in this folder for development

### Key Principles
- **Type Safety:** Use TypeScript for all code
- **Component Reusability:** Create reusable, well-documented components
- **Consistent Styling:** Follow Tailwind CSS patterns
- **Code Quality:** Write clean, maintainable code

## Contributing

When adding new features or making changes:
1. Follow the guidelines in this folder
2. Maintain consistency with existing code
3. Update documentation as needed
4. Test thoroughly before submitting
